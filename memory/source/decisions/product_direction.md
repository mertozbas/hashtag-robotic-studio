# Hashtag Robotic Studio Product Direction

Status: active

Hashtag Robotic Studio is the customer-facing desktop application shipped with Hashtag Robotics SO101 robots.

The app should let a customer connect SO101 USB devices and manage setup, calibration checks, camera preview, teleoperation, dataset recording, dataset review, policy validation, simulation, diagnostics, and agent-driven robot operation from one Turkish-first UI.

The product is not merely a LeRobot UI and not merely a Strands demo dashboard. It is a complete local robot studio that uses LeRobot for SO101 hardware/data truth and Strands Robots plus Strands Agents for orchestration and agent operation.

Core product promise:

- plug in the robot
- detect devices
- verify calibration
- operate safely
- collect data
- validate policies
- let an agent use the robot through explicit tools and safety contracts
