# Glass Cockpit UI Direction

Status: active

Use glassmorphism as a serious robotics cockpit style, not as decorative marketing.

Use glass for:

- top status shell
- side navigation
- camera HUD
- non-critical summaries

Use solid high-contrast panels for:

- emergency stop
- motor lock
- calibration write
- real rollout
- logs
- dense tables
- warnings

Turkish is the default UI language and English is selectable. Critical strings must be concise and unambiguous. The UI should be dense, operational, and scannable.
