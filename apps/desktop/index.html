    Fiziksel hareket kuralı

    Teleop, kayıt, motor testi ve real rollout kullanıcı unlock, süre limiti, acil stop, workspace_clear ve SafetyGate onayı olmadan başlamaz.

    Kalibrasyon dosyasını yazma veya üzerine yazma işlemi backup, robot rol doğrulaması ve açık kullanıcı onayı olmadan kapalıdır.

    Dashboard ekranları

    Başlangıç readiness skorunu, Cihazlar port/kamera adaylarını, Kalibrasyon dosya durumunu, Kayıt dataset planını, Policyler model uyumluluğunu ve Agent tool izinlerini gösterir.

    API key vault

    Hugging Face, OpenAI, Anthropic, Google Gemini, GitHub, W&B, AWS ve Custom MCP keyleri masked status olarak tutulur; secret değerleri response, log veya support bundle içinde gösterilmez.

    Hugging Face eğitim akışı

    Dataset kalite kontrolü tamamlandıktan sonra policy family, compute hedefi ve output model repo seçilir. Eğitim işi kullanıcı onayı olmadan başlatılmaz.

    Agent operasyon modu

    Agent diagnose ve simulation tool'larını kullanabilir; fiziksel tool'lar session scope, operation contract ve SafetyGate olmadan kapalı kalır.
